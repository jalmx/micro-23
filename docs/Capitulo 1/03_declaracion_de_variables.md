# ¿Cómo se declaran las variables ?

Cada lenguaje tiene su sintaxis de declaración de variables, en nuestro que es `lenguaje C`, es la siguiente manera:

Sintaxis:

```c
tipoDeDato nombreVariable = valor;
```

Ejemplos:

```c
int edad = 10;
float altura = 1.4;
char letra = 'a';
bool lleno = true;
```
